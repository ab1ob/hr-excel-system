# نظام HR المتكامل — Excel

نظام موارد بشرية تشغيلي كامل في ملف Excel واحد، للمنشآت الصغيرة والمتوسطة التي لا تحتاج نظاماً سحابياً. البيانات المرفقة تجريبية بالكامل: ثلاثون موظفاً افتراضياً في شركة خيالية.

[⬇️ تحميل النظام (ZIP)](https://github.com/ab1ob/hr-excel-system/archive/refs/heads/master.zip)

## الأوراق

| الورقة | الوظيفة |
|--------|---------|
| لوحة التحكم | مؤشرات المنشأة في شاشة واحدة |
| الإعدادات | قواعد الحساب وبيانات المنشأة من مكان واحد |
| بيانات الموظفين | السجل الكامل: العقود، الهويات، الإقامات، الرواتب الأساسية |
| الحضور | متابعة شهرية للحضور والغياب والتأخير |
| الرواتب | مسير شهري بحسابات تلقائية مرتبطة بالإعدادات |
| الإجازات | الأرصدة والطلبات بأنواعها |
| الخصومات والمكافآت | تسوية شهرية تنعكس على المسير |
| KPI الأداء | مؤشرات أداء لكل موظف |

## الحسابات المطابقة للأنظمة السعودية

- اشتراكات التأمينات الاجتماعية (GOSI) للسعودي وغير السعودي
- مكافأة نهاية الخدمة (EOSB) وفق نظام العمل
- متابعة تواريخ انتهاء الإقامات

## الاستخدام

افتح `نظام_HR_المتكامل.xlsx` وابدأ من لوحة التحكم. استبدل البيانات التجريبية ببيانات منشأتك — الحسابات كلها معادلات تعمل تلقائياً.
